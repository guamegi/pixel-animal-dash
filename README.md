# 🐥 Animal Dash

**Animal Dash**는 귀여운 아트 스타일의 동물 캐릭터를 조작하여 장애물을 피하고 별을 모으며 최고 점수에 도전하는 웹 브라우저 기반 아케이드 게임입니다.

---

## 🎮 게임 특징

- **히어로 선택**: 병아리(🐥), 펭귄(🐧), 부엉이(🦉), 고양이(🐱) 중 나만의 캐릭터를 선택해 플레이할 수 있습니다.
- **레벨 시스템**: 10점마다 레벨이 오르며, 레벨이 높아질수록 속도가 빨라져 난이도가 상승합니다.
- **아이템 수집**: 장애물 사이에 등장하는 별(⭐)과 보석 등을 먹으면 추가 점수를 획득할 수 있습니다.
- **스킬 시스템**: 캐릭터 별 특색있는 스킬을 사용할 수 있습니다.
- **반응형 사운드**: 점프, 아이템 획득, 충돌 상황에 맞춰 실시간으로 생성되는 효과음을 즐길 수 있습니다.
- **안전한 재시작**: 게임 오버 시 2초간의 입력 대기 시간을 두어 실수로 재시작되는 것을 방지합니다.

## 🕹️ 조작 방법

### 캐릭터 선택 화면

- **키보드 방향키 (←, →, ↑, ↓)**: 캐릭터 선택시 칸 이동
- **스페이스바(Space) 또는 엔터(Enter)**: 캐릭터 확정 및 게임 시작
- **마우스**: 캐릭터 클릭 및 'START' 버튼 클릭으로 시작 가능

### 게임 플레이 중

- **스페이스바(Space) 또는 마우스 클릭**: 캐릭터 점프
- **게임 오버 시**: 2초 후 나타나는 화살표 UI를 확인하고 클릭/스페이스바를 눌러 재시작

## 🛠️ 기술 스택

- **HTML5 Canvas**: 게임 그래픽 렌더링
- **JavaScript (Vanilla JS)**: 게임 로직 및 물리 엔진 구현
- **Web Audio API**: 브라우저 내장 오디오 합성 기술을 이용한 효과음 구현
- **CSS3**: UI 레이아웃 및 픽셀 애니메이션

## 🚀 실행 방법

1. 이 레포지토리를 클론하거나 소스 코드를 다운로드합니다.
2. `index.html` 파일을 크롬, 엣지 등 최신 웹 브라우저에서 실행합니다.
3. 캐릭터를 고르고 바로 게임을 시작하세요!

---

**Tip**: 캐릭터가 작아질수록 좁은 틈새를 통과하기 쉬워지지만, 속도가 빨라지니 주의하세요!
